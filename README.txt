# Three.js Projekt-Sammlung

Dies ist eine Sammlung von drei einfachen Three.js-Demos, die verschiedene Funktionen der 3D-Webentwicklung demonstrieren.  
Alle Projekte sind lokal mit dem **Live Server** in **Visual Studio Code** lauffähig.

## Voraussetzungen

- Visual Studio Code
- Erweiterung: **Live Server**
- Ein moderner Browser (z. B. Chrome, Firefox)

## Projekte

### 1. Demo Cube

Ein einfacher, interaktiver Würfel, der sich mit der Maus frei drehen lässt.

- Ziel: Einstieg in die grundlegende Szenenerstellung mit Three.js.
- Features:
  - Interaktive Kamerasteuerung
  - Farbiger 3D-Würfel
  - Orbit Controls zur freien Drehung

### 2. Demo Man

Eine animierte Poly-Figur (importiert von [PolyPizza](https://poly.pizza)), dargestellt in einer Skybox.

- Ziel: Einführung in Modelle, Animationen und Umgebungsdarstellung.
- Features:
  - Importierte 3D-Figur mit Animation
  - Umgebungs-Skybox aus 6 Bildern
  - Beleuchtung und Atmosphäre

### 3. Demo FPS

Ein minimalistisches First-Person-Shooter-Showcase.

- Ziel: Grundlagen von Bewegung und Interaktion im 3D-Raum.
- Features:
  - First-Person-Steuerung
  - Bewegung durch die Szene
  - Schießen kleiner Bälle als Projektile

## Verwendung

1. Projektordner in VS Code öffnen.
2. Rechtsklick auf `index.html` → **"Open with Live Server"**.
3. Die Demo wird im Browser geöffnet und kann sofort verwendet werden.
